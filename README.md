# dotvim

.vim folder containing all plugins and settings for Vim

## Usage
1. `git clone git@github.com:samchandra/dotvim.git`
2. `mv dotvim .vim`
3. `ln -s .vim/.vimrc .vimrc`
4. `mkdir .vim_tmp`
5. `mkdir .vim_tmp/backup`
6. `mkdir .vim_tmp/tmp`

## Removing plugin
Remove the plugin folder from the bundle directory
